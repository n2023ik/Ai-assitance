# Dazzy AI Assistant

A desktop AI assistant built with Python and Tkinter.
Uses DeepSeek API for intelligent responses.

## Features
- Chat-based AI assistant
- Wikipedia search
- Open websites via commands
- Time, date, jokes

## Tech Stack
- Python
- Tkinter
- DeepSeek API

## How to Run
1. Create a `.env` file
2. Add:
   DEEPSEEK_API_KEY=your_api_key
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   python app.py
